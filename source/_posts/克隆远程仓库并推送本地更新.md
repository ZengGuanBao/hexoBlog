# 克隆远程仓库并推送本地更新 #

**1.新建远程仓库（vueMusic）**

![](https://i.imgur.com/hlJZZO8.png)

**2.克隆远程仓库**
- git clone git@github.com:ZengGuanBao/vueMusic.git

![](https://i.imgur.com/Q0gbMz0.png)

**3.有更新的情况操作步骤**
- git status
- git add .
- git commit -m "文件提示"

![](https://i.imgur.com/GTGT6QO.png)

**4.再推送本地库到远程仓库**
- git push origin master
